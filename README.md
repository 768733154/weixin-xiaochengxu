# 怀古小游戏（微信小程序）

一个以复古风格为主题的微信小程序小游戏合集，包含多种经典玩法，并支持五子棋好友联机对战。

## 功能列表

- 俄罗斯方块
  - 80 年代街机风界面
  - 开始/暂停控制
  - 长按左移、右移、下落快速操作
- 贪吃蛇
  - 基础可玩版本
- 2048
  - 基础可玩版本
- 五子棋
  - 人机对战（简单 / 普通 / 困难）
  - 好友联机对战（创建房间、分享邀请、加入房间）
  - 标准交叉点落子、天元标记、边缘半截线

## 项目结构

```text
.
├── app.js
├── app.json
├── app.wxss
├── pages

│   ├── snake       # 贪吃蛇
│   ├── game2048    # 2048
│   └── gomoku      # 五子棋（人机+联机）

```

## 本地运行

1. 打开微信开发者工具
2. 导入项目目录：`/Users/zq/长亭/学习/小程序`
3. 使用小程序模拟器预览与调试

## 五子棋联机配置（云开发）

联机模式依赖微信云开发数据库。

1. 在微信开发者工具中开通云开发环境
2. 创建集合：`gomoku_rooms`
3. 开发阶段可先使用宽松读写权限（发布前请收紧）
