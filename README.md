# 怀古小游戏（微信小程序）

一个以复古风格为主题的微信小程序小游戏合集，包含多种经典玩法，并支持五子棋好友联机对战。

## 功能列表

- 俄罗斯方块
  - 80 年代街机风界面
  - 开始/暂停控制
  - 长按左移、右移、下落快速操作
- 贪吃蛇 2.0（Neon Snake）
  - 现代化视觉、霓虹棋盘、自适应蛇身发光效果
  - 滑动手势 + 虚拟 D-Pad 双操作模式
  - 动态加速、记分面板与本地最高分存档
- 2048
  - 基础可玩版本
- 五子棋
  - 人机对战（简单 / 普通 / 困难）
  - 好友联机对战（创建房间、分享邀请、加入房间）
  - 标准交叉点落子、天元标记、边缘半截线
- 猜灯谜闯关
  - 50 关随机灯谜题库
  - 答错立即失败，结算页展示奖励与本地排行榜
- 古诗词闯关
  - 50 关经典诗句快答，题库独立
  - 共用结算页但记录单独存储，支持重开随机抽题
- 见缝插针
  - 30 关渐进式难度，旋转速度、初始障碍、发射针数量随关提升
  - 点击盘面或按钮发射，失败/通关会写入本地成绩榜

## 本地运行

1. 打开微信开发者工具
2. 导入项目目录：`/Users/zq/长亭/学习/小程序`
3. 使用小程序模拟器预览与调试

## 五子棋联机配置（云开发）

联机模式依赖微信云开发数据库。

1. 在微信开发者工具中开通云开发环境
2. 创建集合：`gomoku_rooms`
3. 开发阶段可先使用宽松读写权限（发布前请收紧）

## 已完成的核心体验优化

- 俄罗斯方块：全屏自适应、复古街机外观、按键交互增强
- 贪吃蛇：现代霓虹风 UI、滑动 + 虚拟按键双控、动态速度与本地最高分
- 猜灯谜 / 古诗词闯关：独立 50 题随机闯关、答错即失败、结算页含奖励与排行榜
- 见缝插针：暖色颗粒美术、旋转演出、30 关递增节奏，并记录本地成绩榜
- 五子棋：
  - 从格子中心落子修正为交叉点落子
  - 补充天元标记和边线半截处理
  - AI 从随机落子升级为攻防评分决策

## 排行榜与存档

- 猜灯谜、古诗词闯关：通关或失败后会写入本地排行榜，记录最高关卡、得分与时间戳
- 见缝插针：每次纪录闯关成绩，展示最佳记录与前五名历史表现
- 贪吃蛇：在本地缓存最高分，随时可查看并挑战
